# AI-Based Traffic Congestion Prediction using Edge AI

## Project Overview

Traffic congestion is one of the major problems in modern urban areas.
Traditional traffic monitoring systems rely on manual observation or basic sensors, which cannot predict congestion in advance.

This project proposes an **AI-based real-time traffic monitoring and congestion prediction system** using existing CCTV cameras and edge computing devices.
The system analyzes live traffic video, detects vehicle density, and predicts congestion using machine learning models.

The solution is designed to work with existing surveillance infrastructure without modifying government protocols.

---

## Objectives

* Detect traffic density using AI
* Predict traffic congestion in real time
* Reduce manual monitoring effort
* Provide alerts for traffic control authorities
* Support smart city traffic management

---

## Features

* Real-time video processing
* Vehicle detection using AI model

---

## System Workflow

1. Capture live video from traffic CCTV
2. Convert video into frames
3. Preprocess frames
4. Detect vehicles using AI model
5. Calculate traffic density
6. Predict congestion level
7. Generate alert if congestion detected
8. Store alert data in database
9. Notify traffic authority
10. Continue monitoring
